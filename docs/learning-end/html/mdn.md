# mdn

#### `<header>` 
标题元素，介绍性内容，通常包含一组介绍性的实用元素

#### `<section>`
流式元素，章节内容

### `<footer> `
尾部元素

#### `<blockquote>`
HTML 块级引用元素，代表其中的文字是引用内容。通常在渲染时，这部分的内容会有一定的缩进，阴影部分，同其他文本区分分辨开来

#### `<b>`
用于吸引读者的注意到该元素的内容上（如果没有另加特别强调）。这个元素过去被认为是粗体（Boldface）元素

#### `<del>`
从文档中删除的文字内容
